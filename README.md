# Zona de Controle (MVP)

Projeto estático (HTML/CSS/JS) para rodar em GitHub Pages.
Modo: celular único, offline (sem backend).

## Arquivos
- index.html
- styles.css
- app.js

## Rodar local
Abra o index.html no navegador.

## Publicar no GitHub Pages
1. Suba estes arquivos para a raiz do repositório
2. GitHub → Settings → Pages
3. Source: Deploy from branch
4. Branch: main e folder /root

## Observações
- O app salva o estado em localStorage
- Botão Reiniciar apaga a sessão e gera um novo protocolo
